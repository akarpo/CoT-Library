# CoT-Library

Analysis of City of Troy, Michigan public library funding (revenue, operating
expenditures, and capital expenditures) as reported in the City's Comprehensive /
Annual Comprehensive Financial Reports (CAFRs / ACFRs), fiscal years
**2000 through 2025**.

Open [`index.html`](./index.html) in a browser to read the full write-up with
embedded year-by-year tables and PDF links.

## What's in this repo

| Path | Contents |
| --- | --- |
| `index.html` | The analysis as a single static page. |
| `reports/` | The City's ACFR / CAFR PDFs, FY2003 – FY2025. |
| (linked out) | FY2000–FY2002 CAFR scans (33–34 MB each) are linked to the City's CMS. |

## Source

All financial reports are published by the City of Troy, MI on the
[Financial Documents page](https://troymi.gov/departments/city_manager/financial_services/financial_documents.php)
of the city website. Files are mirrored here for stable citation; the City's
copies remain authoritative.
